Installation
======================================

To use PyOgmaNeo2, you must first have an installation of `OgmaNeo2 <https://github.com/ogmacorp/OgmaNeo2>`_.

Follow the instructions in the `OgmaNeo2 README <https://github.com/ogmacorp/OgmaNeo2/blob/master/README.md>`_ of the OgmaNeo2 repository to install it.

After this, you may install PyOgmaNeo2 as described in the `PyOgmaNeo2 README <https://github.com/ogmacorp/PyOgmaNeo2/blob/master/README.md>`_.
